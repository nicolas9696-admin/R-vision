Bilan de puissance

Formule fondamentale — examen

Intensité d'introduction

I = (P × fs) / (400 × √3 × cos φ) = (P × fs) / 623.5
  • P = puissance totale installée (W)
  • fs = facteur de simultanéité selon NIBT ou adapté
  • cos φ = 0.9 — standard et minimum admis par les GRD suisses
  • 400 V × √3 = 692.8 → × 0.9 = 623.5
  • Installations PV : NE PAS compter dans le bilan. Si PV > 30 kVA → compteur d'injection obligatoire

Sélectivité — point critique examen

Sélectivité totale, partielle, filiation

  • Sélectivité totale = SEUL le disjoncteur en aval du défaut déclenche dans tous les cas
  • Sélectivité partielle = sélective seulement jusqu'à une certaine valeur de Icc
  • Filiation (back-up) = protection du disjoncteur aval, mais les DEUX peuvent déclencher → pas de continuité garantie
  • Utiliser la tabelle constructeur fournie dans la consigne
  • T = totalement sélectif | – = pas de sélectivité | valeur numérique = sélectif au-dessus de ce Icc
Exemple CERPE : Icc=4.3kA | Groupe B40A | Intro 80A → PAS de sélectivité
→ HPC 100A nécessaire pour sélectivité totale
Exercice complet : 15×C16A mono + 3×C13A tri + 1 CE 9kW tri, fs=0.5
→ P = (15×3.6 + 3×9 + 9) = 54+27+9 = 90 kW
→ I = (90000×0.5)/623.5 = 72.2 A → Introduction 80A

Calcul d'éclairage et TV

Formule — à maîtriser

Nombre de luminaires

N = (Surface [m²] × Éclairement [lux]) / (Flux lumineux [lm] × Rendement η)
  • η sans indication → 0.5 (plage 0.4–0.6)
  • η intègre : rendement local + luminaire + vieillissement
  • Arrondir à l'entier supérieur, puis ajuster pour répartition
Exemple CERPE : 5×6m=30m² | 500lux | 2360lm | η=0.45
N=(30×500)/(2360×0.45)=15000/1062=14.1 → 15 luminaires

Exercice : 8×10m | 300lux | 4000lm | η=0.5
N=(80×300)/(4000×0.5)=24000/2000=12 → 12 luminaires

C13A monophasé et triphasé : quelle puissance dans le bilan ?›

Sélectivité : groupe B40A, Icc=4.3kA, intro 80A. Sélectivité ? Solution ?›
NON, pas de sélectivité. Solution : remplacer l'introduction 80A par un HPC 100A (selon tabelle constructeur).
Citez tous les éléments obligatoires sur un schéma de distribution.›
1. Nom GRD. 2. Système mise au neutre avant+après CSG. 3. Sections et nb conducteurs tous tronçons. 4. Mises à terre (électrode, PE, sections). 5. Protections (type B/C/D, In, DDR). 6. Numéros groupes avec désignation. 7. Éléments GRD : TC, protection TC, récepteur délesté, lignes pilotes. 8. Sections liaisons équipotentielles.
Code couleur plan : éclairage normal / force / secours / KNX ?›

Max tubes/boîte ? Max appareils/groupe ? Prises doubles = combien ?›

M4.1

Différence contrôle final vs contrôle de réception ?›
Contrôle final = entreprise ayant fait les travaux, autorisation I ou K. Contrôle de réception = entreprise N'AYANT PAS participé, autorisation K uniquement, dans les 6 mois, obligatoire si périodicité <20 ans.
5 types de contrôles OIBT avec autorisations.›
1. 1ère vérification = monteur, I. 2. Contrôle final = installateur, I ou K. 3. Contrôle réception = indépendant, K accrédité. 4. Contrôle périodique = indépendant, K. 5. Sporadique/ponctuel = indépendant, K.
Autorisation K : que peut et ne peut pas faire cette entreprise ?›
PEUT : contrôles finaux, périodiques, réception installations standards. NE PEUT PAS : locaux spéciaux (zones Ex, gares, locaux médicaux groupe 2). Ces derniers nécessitent entreprise accréditée ESTI.
Ordre des mesures lors d'un contrôle. Pourquoi continuité PE en premier ?›
1. Continuité PE (sans tension — OBLIGATOIREMENT EN PREMIER). 2. Riso (sans tension). 3. Test DDR (avec tension). 4. Icc. 5. Champ tournant. 6. Tension/fréquence. Raison : le PE est utilisé comme référence pour toutes les autres mesures. Si PE interrompu et non détecté → toutes les mesures suivantes sont faussées = DANGEREUX.
5 défauts NIBT fréquents sur les EA.›
1144 = couvercle manquant. 1162 = accessibilité insuffisante. 1312 = sectionneur neutre manquant. 3540 = schéma ne correspond pas. 3552 = documentation manquante. Aussi : 1282 = marquage Imax HPC absent. 3130-3134 = amiante. 3136 = PCB rouge.
Défauts DDR les plus importants (codes + descriptions).›
3320 = DDR type AC INTERDIT (→type A). 3310 = DDR 30mA manquant prises libres. 3312 = DDR 30mA manquant circuits luminaires habitation (NOUVEAU 2025). 3316 = DDR en TN-C (inadmissible). 3352 = DDR ponté (supprimer immédiatement). 3340 = DDR ne déclenche pas au test (remplacer). 3314 = DDR 300mA (→30mA).
DDR obligatoire en salle de bains ? Et pour une borne VE ?›
SDB : DDR ≤30mA pour TOUS les circuits (NIBT 7.01.4.1.5.1). Borne VE : DDR ≤30mA type B OU type A si borne avec détection DC intégrée >6mA. Chaque point de raccordement son propre DDR (NIBT 7.22.5.3.1.3.101).
2 principales nouveautés DDR dans la NIBT 2025.›
1. DDR type AC INTERDIT (code 3320 → type A minimum obligatoire pour tout nouveau DDR). 2. DDR ≤30mA obligatoire pour circuits luminaires dans pièces d'habitation (NIBT 4.1.1.3.4 — NOUVEAU dans NIBT 2025).

M4.2 — Mesures

Riso : valeur min légale ? Tension test ? Appareils à déconnecter ?›
Riso min = 1MΩ (installations ≥2010). Tension test = 500V DC pour 230/400V. Déconnecter : variateurs, ballasts électroniques, onduleurs, appareils IT, LED avec drivers → risque dommage par 500V DC.
Différence erreur absolue vs relative. Exemple.›
Absolue = valeur fixe inhérente à l'appareil (ex: ±4V sur calibre 400V à 1%). Relative = absolue/valeur mesurée×100% (ex: 4/250 = 1.6%). Plus l'appareil est précis (classe plus élevée), plus l'erreur absolue est faible.
3 étapes test DDR complet + temps de déclenchement admis.›
1. Bouton mécanique → déclenche. 2. 0.4×IΔn → NE DOIT PAS déclencher. 3. IΔn → DOIT déclencher en ≤0.4s pour circuits terminaux ≤32A avec prises.
Types A, B, F de DDR : quelle différence ?›
Type A = AC + courants pulsés continus → minimum obligatoire Suisse. Type F = AC + pulsés + fréquences mixtes ≤1kHz → variateurs monophasés, PAC. Type B = tous courants y.c. DC lisse → onduleurs PV, VE sans détection DC intégrée.
SNR 462638 : ordre de contrôle + valeurs Riso par classe.›
Ordre : 1. Classe de protection. 2. Visuel + fonctionnement. 3. Continuité PE (Cl.I, ≤0.3Ω/5m). 4. Riso (Cl.I ≥1MΩ, Cl.II ≥2MΩ, Cl.III ≥0.25MΩ). 5. Courant PE/fuite (Cl.I ≤3.5mA, Cl.II ≤0.5mA, Cl.III ≤1mA).
RE max pour DDR 30mA système TT ? Et pour 300mA ?›
RE = UB/IΔn = 50/0.03 = 1650Ω pour DDR 30mA. Pour DDR 300mA = 50/0.3 = 165Ω. Pour DDR 10mA = 50/0.01 = 5000Ω.
Quelle méthode mesure de terre est utilisable derrière un DDR ?›
Seule la méthode par boucle (pince de mesure, image 7 ESTI) est utilisable derrière un DDR. Les méthodes courant/tension (image 5) et tension réseau (image 8) NE peuvent pas être utilisées derrière un DDR car elles déclencheraient le DDR.

M4.2 — Systèmes et Terre

Différence TN-C vs TN-S ? Risques TN-C ?›
TN-C = conducteur PEN (N+PE combinés, Sch. III). TN-S = N et PE séparés (standard actuel). Risques TN-C : 1) DDR interdit. 2) Rupture PEN → 115-230V sur masses → mortel (bœuf ESTI). 3) Contrôle 5 ans. 4) Inversion N/PE possible.
Sections conducteur de terre enterré et équipotentialité principale.›
Conducteur de terre enterré : min. 50mm² Cu. Non enterré : min. 16mm², max 50mm². Équipotentialité principale : min. 6mm² Cu (10mm² si foudre), max 25mm², = 0.5× section PE de référence.
Pourquoi la mesure de RE est obligatoire selon l'ESTI 01/2021 ?›
L'électrode de terre est composant essentiel des mesures de protection TN/TT. Sa résistance doit être dans le RS ou PM selon art. 13 O-DETEC. Souvent omise en pratique = non-conformité. Les variations saisonnières peuvent modifier RE de ±50% → mesure nécessaire à chaque nouvelle installation ou modification.
Comment créer une électrode de terre dans un bâtiment existant avec fondation armée ?›
Dégager 2 fers à béton verticaux (≥8mm) dans le mur extérieur le plus proche du sol. Raccorder une languette de connexion aux fers dégagés. Sortir du béton et raccorder au système de terre. Refermer hermétiquement contre la corrosion. (NIBT 5.4.2.2.9, ESTI 09/2012).

M4.2 — IRVE

4 modes de recharge VE + caractéristiques principales.›
Mode 1 = prise standard, 10A max, vélos/motos. Mode 2 = cordon avec ICCPD, prise 230V, 10A. Mode 3 = borne AC dédiée (wallbox), jusqu'à 22kW (32A tri), communication véhicule-borne. Mode 4 = DC rapide, charge directe batterie, 50kW+.
États CP (Control Pilot) et leurs tensions.›
A = +12V/-12V : pas de VE. B = +9V/-12V : connecté, pas prêt → verrouillage borne. C = +6V/-12V : prêt, sans ventilation. D = +3V/-12V : prêt, avec ventilation. E = 0V : court-circuit VE → déclenchement IMMÉDIAT.
Quel DDR pour borne VE ? Différence type A vs type B pour IRVE ?›
DDR ≤30mA obligatoire pour chaque point. Type A suffisant si borne intègre détection DC >6mA. Type B obligatoire si borne sans détection DC intégrée. DDR type B 30mA est le "top" selon cours.
Pourquoi répartir les phases pour plusieurs bornes VE ?›
Certains VE ont des convertisseurs monophasés uniquement. En dessous de 10A certains VE passent automatiquement en mono. Plug-in hybrides = souvent monophasé. Si toutes les bornes sur la même phase → asymétrie de charge (pénalité GRD). Rotation : L1/L2/L3 → L2/L3/L1 → L3/L1/L2.
Procédure contrôle OIBT borne mode 3 AC — étapes principales.›
1. Visuel + docs. 2. Continuité PE + Riso + Icc. 3. Sélectionner ≥20A. 4. Position A → pas réaction. 5. Position B → verrouillage. 6. Position C → enclenchement. 7. Test coupure PE. 8. Vérifier tensions. 9. Icc L-N. 10. Test DDR type A + type B/6mA. 11. Position E → déclenchement immédiat. 12. Rendre opérationnelle.

M4.3 — Sécurité

Seuil fibrillation ventriculaire ? Seuil de lâcher ?›
Fibrillation ventriculaire : possible à 40mA (selon durée), quasi-certaine à 80mA → MORTEL. Seuil de lâcher : 15mA (crispations, impossible de lâcher). Seuil perception : 0.5mA. Rappel : DDR 30mA < 40mA → protection efficace.
Pourquoi 30mA pour les DDR de protection des personnes ?›
30mA est inférieur au seuil de fibrillation ventriculaire (40mA) avec une marge de sécurité. Le DDR coupe avant que le courant atteigne 80mA (seuil mortel). Le temps de coupure 0.4s est aussi critique pour limiter l'énergie transmise au corps.
Condensateur avec autocollant orange : que faire ?›
Autocollant orange = PCB suspecté (code 3137). Informer l'office cantonal des produits chimiques. Le propriétaire a 6 mois pour fournir preuve que teneur PCB <50 ppm. Si impossible → traiter comme confirmé (rouge) = mise hors service immédiate + déchets spéciaux.
5+5 règles vitales SUVA pour travail hors tension.›
1. Mettre hors tension. 2. Condamner (cadenasser). 3. Vérifier absence de tension (VAT). 4. Mettre à la terre et en court-circuit. 5. Délimiter et couvrir pièces voisines sous tension. → Respect des 3 premières = éviterait ~50% des accidents.
Pourquoi hospitaliser une personne électrisée même sans symptômes apparents ?›
Risque de dysfonctionnement des organes vitaux pendant ~50h après électrisation. L'électrolyse du sang peut provoquer coagulation et embolie. La destruction des tissus peut entraîner septicémie. La perte de liquide peut provoquer insuffisance rénale. Surveillance obligatoire en milieu hospitalier.

Questions de synthèse

Remplacement EA en TN-C (Sch. III) : doit-on installer des DDR sur circuits existants non touchés ?›
NON. DDR impossibles en TN-C (NIBT 4.1.1.4.5). Normes de l'époque pour circuits non touchés. MAIS prévoir réserve pour future conversion TN-S. Nouveaux circuits dans le nouvel EA → normes actuelles s'appliquent.
Contenu obligatoire du RS (Rapport de Sécurité).›
Propriétaire + gestionnaire, installateur N° autorisation, organe contrôle N° K, adresse installation, type contrôle, périodicité, protection (fusible, emplacement, In), GRD, N° compteur, N° avis installation, système mise au neutre, signatures (installateur + contrôleur + dates), annexes (liste défauts, PM).
Tableau résistance de terre maximale selon contexte.›
≤0.44Ω = TN/TT + sous-station 30A. ≤1Ω = continuité boucle équipotentielle (référence). ≤1.6Ω = bâtiment avec sous-station 30A. ≤10Ω = protection foudre. ≤20Ω = recommandation générale. ≤1650Ω = DDR 30mA système TT.
3 variantes NIBT 2025 pour coupe-surintensité bornes VE.›
Variante 1 = disjoncteur dans tableau individuel pour chaque borne. Variante 2 = disjoncteur entre câble horizontal et chaque borne. Variante 3 = disjoncteur au départ câble horizontal + limitation numérique du courant via gestion intelligente des bornes.
Quand mesurer RE ? Qui est responsable de la mise à la terre ?›
Mesurer RE : lors de tout nouveau dispositif de mise à la terre ou modification. Avant remblayage/bétonnage (ou photos). Documenter dans RS ou PM. Responsable de la mise à la terre : le PROPRIÉTAIRE de l'installation (pas l'installateur). Travaux sur conduites d'eau remettant en cause la terre → accord propriétaire obligatoire.
Différence sélectivité totale, partielle et filiation (back-up).›
Totale = seul le disjoncteur en aval du défaut déclenche, toujours → continuité garantie. Partielle = sélective jusqu'à une valeur de Icc → au-dessus les deux déclenchent. Filiation = le disjoncteur amont protège l'aval contre surintensités trop élevées → mais les DEUX peuvent déclencher ensemble → pas de continuité garantie.
Contenu obligatoire du classeur de chantier selon cours M3M3.›
Plan hygiène et sécurité (PHS), numéros urgences et adresses, plan d'exécution (à jour), planning intentionnel, bulletins de livraison, notices de montage, PV de chantier. Mis à DISPOSITION DES TRAVAILLEURS sur le chantier — différent du dossier de révision remis au client.

Base à savoir formuler

Analyse de risque

Formule

Risque = probabilité × conséquence

Questions de cahier des charges

  • Quel danger veut-on détecter : intrusion, feu, agression, technique, accès non autorisé ?

Module 3 — Photovoltaïque, IPE, Minergie

PV / IPE

Points à vérifier

  • Puissance onduleur, phases, asymétrie, injection.
  • AI/DRT selon puissance et GRD.
  • Protection DC/AC, sectionnement, parafoudre, signalisation.
  • PV non compté comme charge dans bilan consommation, mais influence raccordement.
Minergie

Principes

  • Enveloppe performante, confort, renouvellement d'air automatique.
  • Énergie renouvelable, PV, monitoring selon exigences.
  • Comparer Minergie, Minergie-P, Minergie-A.
PV côté planification

Dimensionner une chaîne

  • Tension chaîne = nombre de modules en série × tension module.
  • À froid, la tension augmente : vérifier que la tension à vide corrigée ne dépasse pas l'entrée max onduleur.
  • À chaud, la tension baisse : vérifier que la tension MPP reste dans la plage onduleur.
  • Courant string dépend du courant module; en parallèle, les courants s'additionnent.
  • Respecter courant d'entrée max onduleur : dépasser peut créer danger ou bridage selon cas.
IPE / GRD
  • Vérifier mise en parallèle réseau, DRT/AI, compteur production/injection, puissance onduleur.
  • Contrôler asymétrie entre phases, limitation injection si imposée.
  • Prévoir arrêt d'urgence, signalisation, documentation et schéma.
Minergie
  • Minergie vise confort, enveloppe efficace et faible besoin énergétique.
  • Minergie-P renforce l'enveloppe; Minergie-A vise bilan énergétique très performant avec production.
  • Le PV peut compenser une partie du bilan selon exigences du label et règles locales.

Appareil de mesure

Étalonnage et fiabilité

  • Étalonnage légal : au moins tous les 2 ans, à adapter selon l'usage.
  • Contrôleur qui mesure souvent : contrôle/étalonnage plus fréquent, par exemple tous les 6 mois selon pratique de cours.
  • Usage journalier chef de chantier : viser un contrôle annuel.
  • Durée de vie pratique d'un appareil de mesure : dépend de l'entretien; retenir environ 3 à 4 ans pour un appareil très utilisé.
  • La responsabilité de la première vérification reste personnelle : celui qui met sous tension doit pouvoir défendre ses mesures.
Riso pratique

Ce que tes notes ajoutent

  • Mesure d'isolement usuelle : 500 V, environ 1 à 2 mA, car on contrôle au-dessus de la tension de service 3 x 400 V.
  • Industrie 690 V : on peut rencontrer des essais à 1000 V selon installation et appareil.
  • Seuil de perception : 0,5 à 1 mA. Exemple utile : 500 V / 1 MΩ = 0,5 mA.
  • Avec parasurtenseur : commencer prudemment, par exemple 250 V si nécessaire; si le parafoudre est débrochable, mesurer normalement à 500 V.
  • Attention N-PE : une valeur correcte à 250 V peut devenir mauvaise à 500 V si un élément électronique ou parafoudre influence la mesure.
Continuité PE

Procédure à savoir dire

  • Pour la continuité PE : fusible du CSG retiré et sectionneur de neutre ouvert si la procédure l'exige.
  • Courant de mesure typique : 200 mA, tension d'essai 4 à 24 V AC/DC selon appareil.
  • Valeur attendue : très faible, souvent à défendre comme inférieure à 1 Ω ou cohérente avec longueur/section.
  • Tout ce qui est relié au PE doit être contrôlé : conducteurs, bornes, coffrets, masses et carcasses métalliques.
  • Enclencher les points lumineux pour vérifier de bout en bout depuis les bornes du tableau.
Icc, Zi et Rlow

Différence terrain

  • La mesure de court-circuit envoie un courant d'essai important pour être précise, car le cuivre a une résistance très faible.
  • Zi mesure une impédance en alternatif; Rlow mesure une résistance.
  • Sur Metrel / mesure Zi : pas forcément de remise à zéro comme Rlow, car les fils de mesure peuvent être doublés.
  • Rlow nécessite un zéro : les pointes et cordons ajoutent de la résistance.
  • NoTrip L-PE : l'appareil fractionne le courant, par exemple jusqu'à quelques mA, pour éviter le déclenchement du DDR.
Courant de fuite à la pince

Quand c'est acceptable

  • À utiliser comme solution de contrôle si couper l'installation coûte trop cher ou met la production à l'arrêt longtemps.
  • Prendre tous les conducteurs actifs dans la pince; ne pas inclure le PE dans la mesure de fuite globale.
  • Repère de cours : jusqu'à 30 mA = généralement défendable; 300 mA doit être justifié; au-delà, suspecter un défaut d'isolement.
  • La pince ne remplace pas automatiquement une Riso : elle aide à décider si une investigation plus profonde est nécessaire.

Classification et erreurs

  • Analogiques — aiguille, lecture interpolation, sensibles vibrations
  • Numériques — valeur "figée" à l'instant t, plus précis
  • Enregistreurs — mémorisation dans le temps, oscilloscopes

  • Absolue = valeur fixe inhérente (ex: ±4V sur calibre 400V à 1%)
  • Relative = absolue/valeur mesurée×100% (ex: 4/250=1.6%)

Principe et formule

Mesure Icc

  • L'appareil mesure l'impédance de boucle (Zboucle = ZL + ZPE)
  • Calcule Icc présumé via loi d'Ohm
  • Facteur correctif : 0.75 (tient compte variation résistance à chaud)
  • Icc max = mesure au tableau | Icc min = point le plus éloigné
Icc présumé = 0.95 × U0 / Zboucle (U0=230V)
Icc utilisé pour vérification = Icc mesuré × 0.75
⚠️ Mesure Icc IMPOSSIBLE derrière un DDR (le DDR déclenche). Mesurer avant le DDR ou calculer.

Vérification temps de coupure

  • Icc min au point éloigné doit être > 5×In pour courbe B, >10×In pour courbe C, >20×In pour courbe D
  • Si insuffisant → temps de coupure trop long → risque personnes

Sélectivité complète

Total, partiel, filiation

  • Sélectivité totale = SEUL le disjoncteur aval du défaut déclenche, toujours → continuité garantie
  • Sélectivité partielle = sélective jusqu'à une certaine valeur de Icc → au-dessus les deux déclenchent
  • Filiation (back-up) = disjoncteur amont protège l'aval → MAIS les deux peuvent déclencher ensemble → pas de continuité

Comment obtenir la sélectivité totale

  • Utiliser tabelles constructeur (données par le fabricant)
  • HPC amont + disjoncteur aval → souvent sélectivité totale
  • DDR type S amont + DDR instantané aval → sélectivité DDR
  • Différence de 2 calibres minimum entre amont et aval
HPC — Fusibles haute performance

Caractéristiques complètes

  • Courbe temps-courant très rapide pour forts défauts

Non-respect = 50% des accidents électriques

5 règles HORS tension

  1. Mettre hors tension
  2. Condamner (cadenasser, pancarte)
  3. Vérifier absence de tension (VAT)
  4. Mettre à la terre et en court-circuit
  5. Délimiter et couvrir pièces voisines sous tension

5 règles SOUS tension (TST)

  1. Formation spécifique TST obligatoire
  2. Autorisation explicite de l'employeur
  3. EPI adaptés (gants isolants, vêtements arc)
  4. Outillage isolé 1000V
  5. Travailler avec partenaire ou sous surveillance

Définition et obligation

Résistance de terre (ESTI)

  • = Résistance de contact entre l'électrode et le sol

  • Facteurs : humidité, température, profondeur, nature du sol
  • Variations saisonnières : peuvent modifier la valeur de ±50% !
  • Mesure obligatoire et à documenter dans RS ou PM
  • Fréquence non harmoniques utilisées (91–2083 Hz) pour éviter perturbations

Méthode pratique

Vérifier une ligne

  1. Identifier protection : type, calibre, courbe, pouvoir de coupure.

  3. Appliquer facteur de sécurité si demandé par protocole/cours.

  • Filiation : l'amont aide à protéger l'aval, mais les deux peuvent déclencher.
  • La preuve se fait avec tableaux constructeur et données de court-circuit.

Module 5 — Communication

4 oreilles

Schulz von Thun

  • Faits : de quoi parle-t-on objectivement ?
  • Relation : que dit le message sur notre relation ?
  • Révélation de soi : que révèle l'émetteur ?
  • Appel : qu'attend-il de moi ?
Écoute active
  • Questionner ouvertement.
  • Reformuler sans juger.
  • Valider la compréhension.
  • Résumer les décisions.
Shannon-Weaver

Transmission du message

  • Source : produit l'idée ou l'information.
  • Émetteur : encode le message.
  • Canal : voix, téléphone, email, radio, réunion.
  • Bruit : tout ce qui perturbe le message, par exemple mauvaise formulation, bruit ambiant, réseau, stress.
  • Récepteur : décode le signal; destination : personne qui reçoit le message final.
  • Feedback : réponse qui permet de vérifier si le message est compris.
Iceberg

Visible et invisible

  • Visible : mots, informations factuelles, contenu verbal.
  • Invisible : ton, émotions, intentions, valeurs, croyances, contexte relationnel, langage non verbal.
  • Le sens d'un message ne dépend pas uniquement des mots : il dépend surtout de l'implicite.
  • Réflexe : si le ton contredit les mots, clarifier au lieu de supposer.
Écoute active
  • Question ouverte : laisse le choix et ouvre la discussion.
  • Question fermée : vérifie un point précis.
  • Retour/reformulation : valide la compréhension.
  • Objection : traiter calmement, puis orienter vers la suite.
Digital / analogique
  • Digital : mots, phrases, logique, contenu abstrait.
  • Analogique : non-verbal et paraverbal, donc relation et émotion.
  • La communication peut être symétrique ou complémentaire.
Verbal / non-verbal
  • Verbal : les mots.
  • Non-verbal : gestes, mimique, regard, posture, apparence.
  • Paraverbal : ton, rythme, volume, pauses.

Structure

Préparer une information

  • Comment : méthode, moyens, processus.
  • Qui : personnes, responsables, acteurs.
  • Quoi : objet, action, fait, décision.
  • Combien : coûts, quantités, temps, ressources.

À ne pas confondre

  • Dette portable : l'employé doit être informé activement.
  • Dette quérable : l'employé doit aller chercher l'information.
  • Need to have : indispensable, par exemple courant coupé, code d'accès, danger.
  • Nice to have : utile mais non critique, par exemple valeur de tension informative.
Cas chantier important

Plan de communication si le chef est absent

  • Interne : direction adjointe, finance, RH, responsable technique, achats, administration, collaborateurs clés.
  • Externe : consortium, maître d'ouvrage, banque, fournisseurs stratégiques; un interlocuteur unique.
  • Support : réunion de cadrage, compte rendu écrit, tableau de suivi partagé, emails formels archivés.
  • Rythme : J0-J2 cadrage; J3-J10 sécurisation financière/RH; au retour du chef, rapport complet et validation.
  • But : maîtriser risques financiers, continuité de direction, crédibilité externe et stabilité de l'entreprise.
Réunion détaillée

Décider si une réunion est utile

  • Quelle est la plus-value par rapport à un mail ou appel ?
  • Quel est le ratio coût salarial / bénéfice attendu ?
  • Les participants sont-ils préparés et co-responsables ?
  • Faut-il échanger, résoudre, décider ou gérer un conflit ?
  • Si aucun livrable n'est attendu, la réunion est probablement inutile.
Préparation
  • Objectif SMART, participants, présidence, ordre du jour.
  • Gardien du temps, secrétaire, durée, pauses, infrastructure.
  • Fil rouge détaillé et minuté.
  • Convocation claire avec attentes et documents.
Animation
  • Commencer à l'heure, accueillir, clarifier règles et rôles.
  • Recentrer avec l'objectif : que fait-on avec cette information ?
  • Faire participer : tour de table, idées, avis, mesures, feedback.
  • Finir avec décisions, responsables, délais et PV.

Deux axes

Hersey-Blanchard

Maturité du collaborateur

  • D1 : incapable et réticent/incertain -> diriger fortement.

  • Norming : règles et manière de travailler stabilisées.
  • Performing : équipe performante orientée résultat.
  • Compétence clé du chef : donner un objectif clair.

Entretien

Étapes

  2. Accueillir et poser le cadre.
  3. Décrire faits et impacts.
  4. Écouter le collaborateur.

Formulation en JE

  • Observation positive : je décris un fait et son effet.
  • Observation négative : je parle d'un point d'amélioration concret.

Lire la relation

  • Public : ce que je connais de moi et que les autres connaissent aussi.

  • Aveugle : ce que les autres voient mais que je ne vois pas.
  • Inconnu : ce qui n'est connu ni de moi ni des autres.
Évaluation du personnel

Ce qui rend l'évaluation défendable

  • Référentiel de comportements attendus au travail.
  • Objectifs SMART et exemples notés durant l'année.
  • Évaluateurs formés, processus lié à la vision de l'entreprise.
  • Fréquence : fin de période d'essai au minimum, annuel si possible hors discussion salariale, et lors de changement de poste/chef.

CNV

4 étapes

Thomas-Kilmann

5 stratégies

Négociation

Négociation

Préparation selon tes notes

  • Le but est d'obtenir un accord entre deux parties.
  • La réussite dépend d'une bonne préparation, pas seulement de la discussion.
  • Clarifier : difficulté personnelle, relationnelle ou situationnelle ?
  • Penser global et long terme : quelle situation d'avenir est acceptable ?
  • Définir ce que je dois obtenir au minimum et ce que l'autre partie doit obtenir au minimum.
  • Se demander : est-ce l'ego qui parle ou l'intérêt réel ?
MESORE

Plan B

  • Meilleure solution de repli si la négociation échoue.
  • Elle définit le seuil de rupture : jusqu'où je peux céder.
  • Elle donne du pouvoir et évite d'accepter un mauvais accord par peur du vide.
  • Elle se prépare avant la négociation.
IPBM

Comprendre l'autre

  • Intérêts : que veut réellement l'autre ?
  • Préoccupations : que craint-il ?
  • Besoins : quelles nécessités fondamentales ?
  • Motivations : qu'est-ce qui le pousse à agir ?
  • But : trouver un accord gagnant-gagnant.

Module 5 — Gestion de soi, stress et posture de chef de projet

Gestion de soi

Ce qu'on attend d'un chef de projet

  • Prioriser les tâches selon urgence, importance, risque et impact client.
  • Garder une posture calme dans conflit, pression ou imprévu chantier.
  • Identifier ses limites : charge, stress, compétences, temps disponible.
  • Déléguer clairement : objectif, délai, marge de décision, contrôle attendu.
  • Communiquer tôt si un délai, une qualité ou une sécurité est menacée.
Stress

Réponse pratique

  1. Identifier la source : surcharge, conflit, flou, manque de ressources.
  2. Classer : urgent/important, sécurité d'abord.
  3. Découper en prochaines actions concrètes.
  4. Informer les personnes concernées.
  5. Faire un suivi et ajuster.
Pièges

À éviter en cas d'examen oral/cas

  • Dire seulement "je vais parler avec lui" sans structure.
  • Ignorer la sécurité pour tenir un délai.
  • Déléguer sans contrôle ni délai.
  • Réagir émotionnellement sans faits observables.

Stress

Réponse d'examen

  • Le stress est personnel : une même situation ne produit pas la même réaction chez tous.
  • Stress positif : motive, pousse vers l'avant, dynamise.
  • Stress négatif : impact santé, agacement, tension, impression de ne jamais finir.
  • Il révèle souvent une discordance entre besoins individuels et conditions de travail.
  • Réagir : dire stop, analyser la cause, planifier, communiquer, déléguer, garder activité physique et pauses.

  • Travail en hôpital avec déplacements inutiles : planifier zones d'intervention et horaires avec le service technique.
  • Heures de tranquillité, sous-tâches, objectifs courts et action aujourd'hui limitent les pertes.

  2. Informer rapidement le cercle interne utile : direction adjointe, finance, RH, technique, achats, administration.

  4. Mettre en place un dossier centralisé sécurisé avec versionnage et décisions écrites.
  5. Planifier J0-J2 cadrage, J3-J10 sécurisation ressources, puis rapport complet au retour du chef.

Cas examen 2025 : équipe en difficulté

Structure de réponse complète

  1. Observer les faits : communication difficile, moral bas, intégration nouveaux collaborateurs insuffisante.
  2. Sécuriser le cadre : entretien individuel si nécessaire, puis réunion d'équipe cadrée.
  3. Écouter activement : questions ouvertes, reformulation, identifier besoins et obstacles.
  4. Clarifier objectifs communs : rôle de chacun, règles de collaboration, priorités chantier.
  5. Intégrer les nouveaux : référent, explication culture, consignes, suivi court.
  6. Mettre des mesures : tâches, responsables, délais, PV, feedback après 2 à 4 semaines.
  7. Si conflit : CNV, médiation, décisions écrites et escalade RH/direction si nécessaire.

Réponse type : une installation doit respecter les règles techniques reconnues, ne pas mettre en danger les personnes, les choses ou l'environnement, et rester conforme en cas de perturbation prévisible.
Schémas de liaison à la terre

TN, TT, IT

  • TN-S : PE et N séparés. Schéma courant en installations modernes.
  • TN-C : conducteur PEN combiné, restrictions fortes, jamais en aval d'un DDR.
  • TT : prise de terre installation indépendante, protection par DDR essentielle.
  • IT : neutre isolé ou impédant, surveillance d'isolement, continuité de service.
  • Le schéma doit être indiqué sur les schémas de distribution avant/après CSG si nécessaire.
Protection contre les contacts

Base + défaut

  • Protection de base : isolation, enveloppe, obstacle, distance, IP adapté.
  • Protection en cas de défaut : coupure automatique, PE, liaison équipotentielle, DDR.
  • Classe II : double isolation, pas de conducteur PE raccordé à l'appareil.
  • Séparation électrique : un seul récepteur ou conditions strictes.
  • SELV/PELV : très basse tension, transformateur conforme, séparation sûre.
DBO / ensembles d'appareillage

SN EN 61439 — ce qu'on attend au brevet

  • Identifier la norme applicable : SN EN 61439-1 règles générales, parties spécifiques selon type d'EA.
  • Vérifier : tension assignée, courant assigné, tenue court-circuit, degré IP, régime de neutre, échauffement, marquage.
  • Documenter : déclaration de conformité, vérification de conception, vérification individuelle de série.
  • Prévoir réserve, accessibilité, séparation des circuits, repérage, protection contre contacts directs.
  • La filiation/sélectivité dépend des tableaux constructeurs, pas d'une intuition.
À réciter

Influences extérieures et IP

  • Choisir le matériel selon environnement : humidité, poussière, chocs, température, incendie, corrosion.
  • IP2X : protection contre accès au doigt et corps solides >12,5 mm.
  • IPX4 : protection contre projections d'eau, utile en extérieur ou zones humides selon cas.
  • La protection mécanique et l'accessibilité doivent être adaptées à l'usage réel.
Questions rapides

Réponses attendues

  • Pourquoi pas de fusible dans le PE ? Le PE ne doit jamais être interrompu.
  • Pourquoi sectionner le N ? Selon schéma et appareil, mais ne jamais interrompre le PE.
  • Qui est responsable de la sécurité ? Propriétaire/exploitant pour maintien, installateur pour exécution conforme.
  • Quel réflexe devant un défaut ? Danger, article/règle, correction, contrôle après correction.

Phases projet

Logique SIA 108

  1. Définition des objectifs : besoin, contraintes, faisabilité.
  2. Études préliminaires : variantes, coûts approximatifs.
  3. Projet : avant-projet, projet, autorisations, coordination.
  4. Appel d'offres : descriptif, soumission, comparaison, adjudication.
  5. Réalisation : plans d'exécution, chantier, métrés, mise en service.
  6. Exploitation : dossier révision, maintenance, garantie, décompte final.
CAN / CFC / Métré

À ne pas confondre

  • CFC classe les coûts par groupe de construction.
  • CAN décrit les prestations de manière normalisée.
  • Métré mesure les quantités réelles ou projetées selon règles convenues.
  • Le descriptif CAN est considéré prêt à la mise en fonction sauf mention contraire.
  • Le métré contradictoire sécurise la facturation et limite les conflits.
Calculation

Chemin de calcul d'une offre

Prix de revient = matériel + main-d'œuvre + frais + sous-traitance + risques
Prix de vente = prix de revient + frais généraux + marge/bénéfice + TVA si applicable
  • La calculation n'est pas une course au prix le plus bas : elle doit couvrir coûts, risques et capacité réelle.
  • Tenir compte : temps de montage, difficulté, hauteur, accès, coordination, stockage, outillage, déplacements.
  • Le renchérissement doit être clarifié contractuellement lorsqu'il y a délais ou variation de coûts.
  • La CCT influence salaires, horaires, vacances, indemnités, maladie, obligations minimales.
Piège de métré

Prestations souvent oubliées

  • Déchargement, réception, entreposage, distribution, déballage, évacuation emballages.
  • Essais, réglages, aide à mise en service et contrôle de fonction.
  • Hauteur de pose et moyens d'accès au-delà des hypothèses usuelles.
  • Coordination avec autres corps de métier et adaptations chantier.
Réponse type

Comparer deux offres

  • Comparer le périmètre exact, pas seulement le total.
  • Vérifier quantités, exclusions, variantes, délais, conditions de paiement.
  • Contrôler conformité CAN/SIA, normes, garanties, références.
  • Identifier risques : prestations non comprises, prix anormalement bas, réserves.

Bilan puissance

Réflexe de calcul

I = (P × fs) / (400 × √3 × cos φ) avec cos φ souvent 0,9 → diviseur 623,5
  • PV/IPE : ne pas compter comme consommation dans le bilan, mais vérifier refoulement, annonce, compteur injection.
  • Choisir l'introduction selon courant calculé, marge, sélectivité, prescriptions GRD.
  • Prises doubles/triples : attention au comptage selon méthode appliquée.
  • La tabelle constructeur prime pour sélectivité/filiation.
Schéma / EA

Checklist dessin

  • GRD, CSG, compteur, TI si nécessaire, récepteur de télécommande, pilotes.
  • Régime de neutre, PE, électrode, équipotentialité, sections et nombre de conducteurs.
  • Protections : type, calibre, pouvoir de coupure, DDR, courbes, coordination.
  • Face avant EA : échelle, accessibilité, réserve, hauteur, repérage.
KNX

Points clés

  • Bus 30 V SELV, câble 2 fils, topologie ligne/zone/réseau.
  • Max typique : 64 participants par ligne, 15 lignes par zone, 15 zones.
  • ETS pour adressage, paramétrage, diagnostic.
DALI

Éclairage adressable

  • 2 fils non polarisés, dédié éclairage, communication bidirectionnelle.
  • 64 participants par segment, 16 groupes, 16 scènes.
  • Variation exigée ou utile : DALI devient souvent la solution propre.
Éclairage

Formule

N = Surface × lux / (flux luminaire × rendement η)
  • Rendement sans indication : prendre une hypothèse réaliste et l'annoncer.
  • Bureau : ordre de grandeur 500 lux.
  • Vérifier répartition, éblouissement, secours, SIA 387/4 si concerné.
Télématique

Réponses courtes

  • IP : adresse, masque, passerelle, DNS; distinguer réseau et hôte.
  • xDSL : transmission sur paire cuivre, débit dépend distance/qualité ligne.
  • Fibre : monomode/multimode, connectique, rayon de courbure, propreté.
  • VoIP : dépend réseau IP, QoS, alimentation, sécurité, disponibilité.
Sécurité / AV

Analyse de risque

Risque / criticité = probabilité × conséquence
  • Cahier des charges : besoin, périmètre, zones, scénarios, alarmes, utilisateurs, maintenance.
  • Vidéosurveillance : objectif légal/proportionné, zones, stockage, droits d'accès.
  • Détection incendie : norme/projet, zones, alarmes, transmission, asservissements.

Phrase type gagnante

Module 5 — Leadership, communication et gestion du personnel

Logique d'examen Module 5

Répondre en actions concrètes

  • Partir du cas : personnes, objectif, urgence, risque, relation, cadre légal.
  • Proposer une démarche en étapes : préparer, écouter, clarifier, décider, suivre.
  • Nommer les outils : écoute active, questionnement, feedback, CNV, Thomas-Kilmann, Maslow, leadership situationnel.

Communication

Canaux et 4 oreilles

  • Communiquer = verbal, non verbal, para-verbal, contexte.
  • Modèle des 4 oreilles : factuel, relation, révélation de soi, appel.
  • Écoute active : reformuler, questionner, valider, résumer.
Information

Sens unique

  • Une information est une communication à sens unique.
  • Critères : juste destinataire, contenu clair, moment, canal, confidentialité, traçabilité.
  • Dette portable : je transmets; dette quérable : l'autre doit aller chercher.
Réunion

Préparer et conduire

  • Objectif, ordre du jour, participants, durée, rôles.
  • Rôles : animateur, gardien du temps, rédacteur PV, participants.
  • PV : décisions, tâches, responsables, délais.
Feedback / évaluation

Structure d'entretien

  1. Préparer faits observables et objectif.
  2. Accueillir, poser cadre, expliquer intention.
  3. Décrire faits sans jugement.
  4. Écouter la version du collaborateur.
  5. Fixer objectifs SMART et moyens.
  6. Convenir suivi et date.
Conflit / négociation

Traiter avant escalade

  • Résistance = désaccord à traiter avant conflit.
  • CNV : observation, sentiment, besoin, demande.
  • Thomas-Kilmann : compétition, collaboration, compromis, évitement, accommodement.
  • Négociation : objectifs, marge, intérêts, alternatives, critères objectifs.
Motivation / leadership

Adapter le style

  • Maslow : besoins physiologiques, sécurité, appartenance, estime, accomplissement.
  • Leadership situationnel : adapter direction/soutien au degré d'autonomie et compétence.
  • Motiver = sens, reconnaissance, autonomie, progression, conditions correctes.
Recrutement / intégration

Processus propre

  • Définir besoin et profil, sélectionner objectivement, conduire entretien structuré.
  • Intégration : accueil, sécurité, outils, référent, objectifs, feedback rapide.
  • Respecter confidentialité et LPD pour données personnelles.
